## Inclusion in IM projects

Run `composer require infyomlabs/adminlte-templates:dev-develop`

Add this entry in `composer.json` :

```
"repositories": [
        {
            "type": "vcs",
            "url": "git@github.com:infernalmedia/adminlte-templates.git"
        }
    ],
```

Enjoy !
